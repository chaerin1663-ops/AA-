## ✨ Key Features (핵심 기능)
이 프로젝트는 **횡스크롤 자동 달리기를 기본으로 하여 점프를 통해 장애물을 회피하고 아이템을 수집하여 고득점을 획득하는 러닝액션게임**을 중심으로 구현되었습니다.


* **Core Mechanics**:
    * 플레이어의 터치 또는 키 입력(Input.GetButtonDown)에 따라 **점프(Jump)**라는 한 가지 액션만을 허용
    * 점프 입력 시, Rigidbody2D에 **Y축 방향의 순간적인 힘(AddForce)**을 가해 도약
    * 획득한 아이템 수에 따라 점수 변수를 증가시켜 최종 랭킹 점수를 산출합니다.
